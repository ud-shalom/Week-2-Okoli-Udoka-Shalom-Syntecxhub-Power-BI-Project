### 📌 PROJECT OVERVIEW#ProjectOverview
This repository contains the analysis and dashboard developed for the Week 2 project. It focuses on website traffic metrics, user behavior, and key performance indicators (KPIs) to provide actionable data insights.

### 🧹 DATA CLEANING#DataCleaning
* **Data Type Validation**: Verified all data types for accuracy in Power Query.
* **Decimal Formatting**: Standardized decimal places for metrics like "session duration" and "bounce rate."
* **Dataset Preparation**: Processed the website traffic dataset to ensure it matches the dashboard requirements.

### 📊 DAX MEASURES#DAXMeasures
The following measures were created to drive the project insights:
Total Sessions = SUM('Website Traffic'[Sessions])

Total Page Views = SUM('Website Traffic'[Page Views])

Average Bounce Rate = AVERAGE('Website Traffic'[Bounce Rate])

Average Conversion Rate = AVERAGE('Website Traffic'[Conversion Rate])
